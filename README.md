## API

### Markers `/api/mark`

#### GET Markers

- All markers in database
	- GET `/api/mark/all`

- For a given bus angle 
	- `/api/mark/angle/:angle?`

- For a given path ID 
	- `/api/mark/:path_id?`



#### POST Markers

- Save new marker to database
	- `POST /api/mark/save`
